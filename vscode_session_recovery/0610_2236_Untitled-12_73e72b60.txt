5. ByP → Fwd RENAME MAP
------------------------------------------------------------
Scheme: Fwd (PascalCase ident segments) / fwd (snake_case parts) / FWD (macro/uppercase). EXCEPTION codes: KEEP-AS-IS, CLI-DO-NOT-RENAME.

Concept distinction (user explicit):
- BYPASS-FORWARDING (cross-level packet forwarding when a level is bypassed by the predictor) → RENAME to Fwd.
- TRUE replacement-policy bypass (no-fill / way==LLC_WAY victim) → KEEP-AS-IS.

KEEP-AS-IS — TRUE replacement-policy bypass: `LLC_BYPASS` define, gated regions cache.cc:345 + cache.cc:555. Cite (cache.cc:347): `if ((cache_type == IS_LLC) && (way == LLC_WAY)) { // this is a bypass that does not fill the LLC` — replacement-policy concept, independent of forwarding predictor.
